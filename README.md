Micha1207's shell. 
Licensed under GNU GPLv3 (see LICENSE)

Tested on Debian GNU/Linux 13 "Trixie"

### 1. Compiling
Make sure to have installed  the `make` command on your system.

1. Clone repo: `git clone https://github.com/Micha1207/MSH.git`
2. Go to `MSH`: `cd MSH`
3. Then run: `make`. This will make executable `msh` file.
4. Run `msh` executable: `./msh`

### 2. Using MSH
When in MSH (see 1.4.)

1. To list available commands: `help`.
2. To exit MSH: `exit`.
